🥖 PanOrderRD
PanOrderRD es una aplicación web diseñada para digitalizar y optimizar la gestión de pedidos de Doña Lucía, quien se dedica a la elaboración de pan casero, bizcochos y dulces por encargo.

El sistema reemplaza el uso de libretas físicas por una plataforma centralizada que permite un control más eficiente de la producción, los pedidos y las finanzas.

📋 Problema
Actualmente, los pedidos se gestionan a través de WhatsApp y se registran manualmente, lo que provoca:

Falta de organización: Dificultad para visualizar la cantidad total de pedidos en períodos específicos (por ejemplo, fines de semana).
Problemas de inventario: No es posible calcular con precisión los ingredientes necesarios para la producción.
Descontrol financiero: Confusión en el seguimiento de anticipos, pagos completados y saldos pendientes.
✨ Solución
PanOrderRD propone un sistema digital que centraliza toda la información y automatiza los procesos clave del negocio.

⚙️ Funcionalidades
🧾 Gestión Operativa
Registro de clientes y productos en una base de datos centralizada.
Control de pedidos con detalles como fecha de entrega, monto total y estado de pago.
Cálculo automático de insumos basado en los pedidos registrados.
📊 Visualización y Análisis
Vista semanal de pedidos para facilitar la planificación.

Panel (dashboard) con métricas clave:

Total de pedidos
Montos por cobrar
Anticipos recibidos
🔐 Seguridad y Validación
Sistema de autenticación mediante correo electrónico y contraseña.
Validación de formularios en tiempo real con mensajes claros y accesibles.
🛠️ Tecnologías
Frontend: React.js / Vite
Estilos: CSS3 / Tailwind CSS
Backend / Base de datos: MySQL
🎯 Objetivo
Mejorar la organización, reducir errores manuales y facilitar la toma de decisiones en el negocio de Doña Lucía mediante el uso de herramientas digitales.

📚 Aprendizaje
Este proyecto permite aplicar conocimientos de desarrollo web, bases de datos y control de versiones, integrando frontend y backend en una solución real.
